# NativeSharpLzo

[![Nuget](https://img.shields.io/nuget/v/NativeSharpLzo?style=for-the-badge&logo=nuget)](https://www.nuget.org/packages/NativeSharpLzo/)
[![GitHub release (latest by date including pre-releases)](https://img.shields.io/github/v/release/BigBang1112/NativeSharpLzo?include_prereleases&style=for-the-badge&logo=github)](https://github.com/BigBang1112/NativeSharpLzo/releases)

A very simple wrapper around the lzo C library by Markus F.X.J. Oberhumer.

Current lzo version: **2.10**

Supported runtimes:
- `win-x64`
- `win-x86`
- `linux-x64`
- `linux-arm`
- `linux-arm64`
- `browser-wasm`
